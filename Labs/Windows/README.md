# Labs de Windows

En esta sección se documentan ejercicios prácticos relacionados con soporte IT en entornos Windows.

## Objetivo
Practicar la resolución de incidencias comunes de usuarios y sistemas Windows, siguiendo una metodología de troubleshooting paso a paso.

## Temas tratados
- Conectividad de red
- Servicios de Windows
- Usuarios y permisos
- Actualizaciones
- Problemas de rendimiento

## Formato de los labs
Cada laboratorio incluye:
- Descripción del problema
- Procedimiento de diagnóstico
- Solución aplicada
- Aprendizajes clave
