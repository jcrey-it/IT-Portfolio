# Labs de Linux

Laboratorios prácticos orientados a la administración básica y troubleshooting en sistemas Linux.

## Objetivo
Adquirir soltura en el uso de Linux a nivel de soporte técnico y sistemas.

## Temas tratados
- Sistema de archivos
- Permisos y propietarios
- Servicios y procesos
- Logs
- Comandos básicos de red

## Enfoque
Los ejercicios están pensados desde un punto de vista práctico, similar a situaciones reales en un entorno IT.
