# Labs de Redes

Ejercicios prácticos relacionados con redes y conectividad.

## Objetivo
Entender y diagnosticar problemas comunes de red en entornos corporativos.

## Temas tratados
- IP, máscara y gateway
- DNS y DHCP
- Ping y traceroute
- Diagnóstico de conectividad

## Metodología
Se simulan problemas de red y se documenta el proceso de análisis y resolución.
