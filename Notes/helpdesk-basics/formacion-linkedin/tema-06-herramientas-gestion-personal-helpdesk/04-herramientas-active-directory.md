# Herramientas de Active Directory para la gestión de usuarios

En entornos corporativos, los técnicos de HelpDesk necesitan administrar usuarios, grupos y equipos mediante Active Directory. Conocer estas herramientas es fundamental para resolver problemas habituales de acceso y gestión de cuentas.


## 1. Acceso a Active Directory
Para realizar tareas administrativas en Active Directory, es necesario acceder a las herramientas correspondientes.

Situación habitual:
- En un servidor, se accede a través del **Administrador del servidor**.
- Desde ahí se encuentran las distintas utilidades de Active Directory.

Estas herramientas permiten gestionar de forma centralizada los recursos de la organización.


## 2. Usuarios y equipos de Active Directory
Una de las herramientas más utilizadas es **Usuarios y equipos de Active Directory**.

Desde esta consola se puede:
- Visualizar la estructura del dominio.
- Navegar por las carpetas (OU) que contienen:
  - Usuarios
  - Grupos
  - Equipos
- Identificar los **controladores de dominio**.

Esta organización facilita la administración y el control del entorno.


## 3. Gestión de cuentas de usuario
Active Directory permite resolver muchos problemas comunes relacionados con usuarios.

Acciones habituales:
- Restablecer contraseñas de usuarios con problemas de inicio de sesión.
- Desbloquear cuentas bloqueadas.
- Crear nuevos usuarios.
- Crear y administrar grupos.

Estas acciones son fundamentales para el soporte diario en HelpDesk.


## Conclusión
El uso de Active Directory para la gestión de usuarios y equipos permite a los técnicos de soporte resolver incidencias de forma rápida y centralizada. Dominar estas herramientas es clave para ofrecer un soporte eficiente y mantener el correcto funcionamiento del entorno corporativo.
