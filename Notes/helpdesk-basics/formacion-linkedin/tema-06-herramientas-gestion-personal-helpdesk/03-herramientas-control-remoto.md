# Herramientas de control remoto

Las herramientas de control remoto permiten al personal de HelpDesk admnistrar y solucionar problemas en equipos de usuarios sin necesidad de estar físicamente presentes. Estas herramientas pueden usarse tanto para admnistración remota como para acceso directo al escritorio del usuario.


## 1. Administración remota mediante Active Directory
Desde Active Directory es posible administrar usuarios, equipos, grupos y otros objetos sin conectarse directamente al equipo del usuario.

Acciones disponibles:
- Acceder a la **Administración de equipos** de forma remota.
- Revisar el **Visor de eventos** del equipo del usuario.
- Acceder al **Programador de tareas**.
- Gestionar **carpetas compartidas**.
- Revisar información de **rendimiento** del sistema.

Estas acciones permiten diagnosticar y resolver muchos problemas sin necesidad de iniciar sesión en el equipo del usuario.


## 2. Escritorio remoto (Remote Desktop Protocol)
En algunos casos, será necesario conectarse directamente al equipo del usuario.

Características:
- Uso de **Remote Desktop Connection (RDP)**.
- Conexión mediante nombre del equipo o dirección IP.
- Posibilidad de iniciar sesión como el propio usuario o como administrador.
- Acceso completo al sistema, igual que si el usuario hubiera iniciado sesión localmente.

Esta opción es útil cuando se requiere una intervención directa sobre el sistema.


## 3. Herramientas de control remoto de terceros
Existen múltiples soluciones de terceros para acceso remoto.

Ejemplos:
- LogMeIn Rescue
- TeamViewer
- GoToMyPC

Estas herramientas suelen ofrecer acceso sencillo y rápido a equipos remotos a través de internet.


## 4. Quick Assist (Asistencia rápida)
Quick Assist es una herramienta integrada en Windows 10 y Windows 11.

Características principales:
- Funciona a través de internet o red local.
- Requiere que tanto el técnico como el usuario ejecuten la aplicación.
- No está disponible para servidores.
- Proporciona un código que debe compartirse para permitir el acceso.

Funciones adicionales:
- Barra de herramientas con opciones de anotación.
- Chat integrado con el usuario.
- Selección de pantalla en equipos con múltiples monitores.

Esta herramienta es especialmente útil para soporte a usuarios finales de forma rápida y segura.
