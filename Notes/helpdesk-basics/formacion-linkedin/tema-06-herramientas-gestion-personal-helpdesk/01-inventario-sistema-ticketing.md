# Inventario en sistemas de ticketing

Los sistemas de ticketing no solo sirven para crear y administrar tickets, sino que también pueden utilizarse para gestionar recursos y activos dentro de la organización.


## 1. Administración de dispositivos
El sistema de tickets puede usarse para registrar y controlar los dispositivos del inventario.

Opciones: 
- Ejecutar un agente de escaneo para obtener información de los dispositivos automáticamente
- Ingresar los datos de manera individual o manual si es necesario

Esto permite tener un registro actualizado de todos los equipos bajo gestión.


## 2. Administración de software
Además del hardware, se puede administrar el software instalado en cada dispositivo.

Opciones:
- Usar un agente de recolección para recopilar todo el software instalado en una computadora
- Registrar manualmente software adicional si fuera necesario

Esto facilita la auditoría de licencias y la gestión de actualizaciones.


## 3. Otros recursos
Los sistemas de tickets pueden almacenar información adicional que optimiza la operación de HelpDesk, tales como:
- Inventario general
- Recursos internos
- Contraseñas y credenciales seguras
- Procedimientos y documentación

La integración de inventario y tickets centraliza la información y mejora la eficiencia del soporte técnico.
