# Problemas de hardware

Los problemas de hardware pueden deberse a fallos físicos o a problemas de software que se manifiestan como fallos de hardware.


## Herramientas de diagnóstico

### 1. Administración de equipos (Computer Management)
Esta herramienta incluye varias utilidades:

- **Visor de eventos:** Permite revisar los registros del sistema y filtrar errores para identificar la causa de los problemas.
- **Administrador de dispositivos:** Permite habilitar o deshabilitar dispositivos, buscar actualizaciones de drivers y comprobar el estado de cada componente.
- **Administrador de discos:** Permite ver los discos conectados, su estado (en línea o fuera de línea) y crear o modificar particiones entre otras cosas.


### 2. Administrador de tareas
Proporciona información en tiempo real sobre los recursos principales del sistema:

- CPU
- Memoria RAM
- Disco duro
- Tarjeta de red
- GPU (si dispones de ella)

Incluye el **Monitor de recursos**, que permite identificar qué procesos consumen más recursos de hardware (CPU. disco, memoria o red).
