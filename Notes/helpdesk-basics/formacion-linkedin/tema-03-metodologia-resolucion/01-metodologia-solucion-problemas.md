# Metodología de solución de problemas

Cisco desarrolló una metodología estructurada para la resolución de problemas, diseñada para ayudar al soporte técnico a abordar incidencias de forma eficiente y ordenada.

Esta metodología puede adaptarse a las necesidades de cada organización.


## 1. Identificación del problema
El primer paso consiste en identificar el problema y determinar su alcance.

Ejemplo:
Un usuario informa que un software dejó de funcionar. Tras analizarlo, se descubre que el problema afecta a todos los usuarios de su departamento, pero no a toda la empresa.

Esto permite clasificar correctamente la prioridad del incidente.


## 2. Establecer una teoría
Una vez identificado el alcance, se formula una teoría sobre la causa del problema.

Acciones habituales:
- Realizar preguntas al usuario
- Solicitar capturas de pantalla
- Conectarse remotamente al equipo
- Consultar la base de datos del proveedor

En este caso, se detecta que el error ocurre tras una actualización reciente del software.


## 3. Probar la teoría
La incidencia se escala al equipo correspodiente (por ejemplo, desarrolladores).

El equipo:
- Reproduce el error
- Verifica que el problema aparece tras aplicar el parche
- Comprueba que el software funciona correctamente en sistemas sin actualizar

Esto confirma la teoría inicial.


## 4. Plan de acción
Se contacta con el proveedor del software para informar del problema.

El proveedor confirma que:
- Otros clientes han reportado el mismo error
- Se está trabajando en una solución
- Se proporciona un nuevo archivo de actualización


## 5. Verificar la solución
El nuevo archivo es probado por el equipo técnico.

Una vez validado:
- Se instala en el equipo del usuario
- Se confirma que funciona correctamente
- Se verifica que el problema queda resuelto en todo el departamento


## 6. Documentación
Finalmente, se documenta el incidente:

- Descripción del problema
- Causa raíz
- Solución aplicada
- Ubicación del archivo de actualización

Esta información se añade a la base de datos para futuras incidencias similares.
