# IT Portfolio

Este repositorio recoge mi aprendizaje práctico en soporte IT, sistemas y redes, mediante laboratorios propios y documentación técnica.

## Objetivo
Adquirir las bases necesarias para trabajar como Técnico IT / Helpdesk y evolucionar progresivamente hacia ciberseguridad y pentesting.

## Contenido
- Labs prácticos de Windows, Linux y redes
- Notas técnicas y apuntes
- Scripts sencillos de automatización y soporte

## Tecnologías
- Windows
- Linux
- Redes (TCP/IP, DNS, DHCP)
- Troubleshooting

## Metodología
- Aprendizaje práctico mediante laboratorios
- Documentación clara y estructurada
- Enfoque en resolución de problemas reales
